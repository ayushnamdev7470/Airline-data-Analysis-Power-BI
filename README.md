# Airline Data Analysis Using Power BI

## Project Overview
This project focuses on analyzing airline data using Power BI. The objective was to clean raw datasets, build relationships between tables, and create interactive dashboards to generate meaningful insights about airline performance and passenger trends.

## Dataset Information
The project uses multiple datasets:
- Flight Information
- Passenger Information
- Ticket Information

These datasets were cleaned and transformed using Power Query before building the data model.

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel

## Key Tasks Performed
- Data Cleaning and Preparation
- Handling Missing Values and Duplicates
- Data Type Formatting
- Creating Table Relationships
- Data Modeling
- Building Interactive Visualizations

## Key Visualizations
- Passenger Count by Airline
- Booking Status Analysis
- Flight Performance Overview
- Destination-wise Analysis

## Insights Generated
- Identified airlines with highest passenger traffic
- Analyzed booking status distribution
- Evaluated flight performance trends

## Conclusion
This project demonstrates practical implementation of data cleaning, modeling, and visualization techniques using Power BI to support data-driven decision making.
